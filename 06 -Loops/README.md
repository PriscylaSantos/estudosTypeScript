### Estruturas de repetição

#### while
```typescript
while(condition)
{
    //executa enquanto a expressão seja verdadeira(true)
}
```

#### for
```typescript
for((initial_count_value; termination-condition; step)
{
    //codigo
}
```

#### for...in
```typescript
for (var val in list)
{
    //codigo
}
```

#### do...while
```typescript
do 
{
    //codigo

} while(condition)
```

#### break
```typescript
while(condition)
{
    //codigo

    if (boolean_expression)
    {
        //codigo

        break
    }
}
```

#### continue
```typescript
for((initial_count_value; termination-condition; step)
{
    //codigo

    if (boolean_expression)
    {
        //codigo

        continue
    }
}
```

#### loop infinito
```typescript
for(;;)
{
    //codigo
}
```
```typescript
while(true)
{
    //codigo
}
```
